Note to the Reader

Readers will find the datasets used in this monograph below. You can open tables of each dataset or download the file using the appropriate buttons. The data is downloaded as geojson files–a common file for storing data with geographic coordinates and other geographic information. Geojson files are not very readable, so you probably want to convert it into a csv file at a website such as data.page or anyconv.com.
